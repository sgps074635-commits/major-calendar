# 重大行事曆系統 - AntiGravity 專案規則與設定

本專案是一個無 GAS、基於 Vite + Vue 3 的重大行事曆系統。

## 技術棧 (Tech Stack)
* **核心框架**：Vue 3 (Composition API / SFC)
* **開發工具**：Vite
* **樣式系統**：精緻 Vanilla CSS (客製化變數、暗色模式、Glassmorphism、微動畫)
* **儲存系統**：
  - 本地模式：`localStorage` (自動儲存)
  - 網路/備份：JSON 備份匯入/匯出、Excel/CSV 匯出。
  - (可擴充) 雲端模式：Firebase Firestore

## 專案核心規則與限制
1. **完全脫離 GAS**：不得使用 `google.script.run`，資料讀寫完全由前端 API 服務層或本機 LocalStorage 處理。
2. **設計美感要求**：
   - 介面必須有高級感、現代感（包含毛玻璃效果、細緻微動畫、柔和色彩、自訂字體如 Outfit/Inter）。
   - 欄位寬度調整後須持久化保存至 `localStorage`。
   - 支援完整的深色模式 (Dark Mode)。
3. **安全防護**：
   - 進入「編輯模式」時需輸入密碼。預設密碼為 `admin123` (可由使用者於設定中自訂變更)。
   - 一般使用者在未登入狀態下僅能使用「檢視模式」與「匯出列印」功能。
4. **離線/本機可用性**：
   - 所有第三方庫如圖標與字型，應儘量在打包時本地化，避免在完全無網環境下失效。
